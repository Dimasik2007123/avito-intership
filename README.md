# Тестовое задание для стажёра Frontend (весенняя волна 2026)

Веб-приложение — личный кабинет продавца с интегрированным AI-ассистентом, который помогает улучшать описания объявлений. Продавец видит список своих объявлений, выбирает нужное и может просмотреть карточку товара, перейти к редактированию и запросить рекомендации от нейросети. Сервис анализирует содержимое карточки товара и предлагает улучшения текста.

### Реализованные дополнительные возможности

- Возможность запустить клиент и сервер через docker compose
- Переключатель layout: сетка/список

### Принятые решения

- для корректного перехода на страницу с детталями товара в функцию получения карточек на сервер был добавлен возврат id товара
- для возможности взаимодействия с сервером при запуске через Docker в метод listen сервера был передан параметр host: "0.0.0.0"
- для реализации возможности перехода со страницы редактирования и просмотра деталей товара к спискам карточек блок с логотипом Авито и адресом сайта был обернут в ссылку, ведущую на главную страницу
- в макете представлены различные размеры кнопок запросов к нейросети: в блоке с инструкцией их ширина составляет 195 пикселей, а в самом интерфейсе приложения размеры иные. Для обеспечения единообразия стилей кнопкам генерации описания и цены была установлена фиксированная ширина в 195 пикселей

### Как запустить проект

Клонировать репозиторий.
Находясь в корневой папке проекта, выполнить команду (требуется установленный Docker)

```
docker-compose up -d
```

### Запуск без Docker

```
# Бэкенд
cd server
echo "PORT=8080" > .env
npm install
npm start

# Клиент
cd client
npm install
npm run dev

# Ollama (требуется установка)
ollama pull llama3
ollama serve
```
